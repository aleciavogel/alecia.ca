## Running the project locally

Install [Yarn](https://yarnpkg.com/) if you haven't already.

```bash
# Clone the repository
git clone git@github.com:aleciavogel/alecia.ca.git
cd alecia.ca

# Install dependencies
yarn 

# Install dependencies in custom plugin
cd plugins/gatsby-plugin-social-banners
yarn

# Start the dev server
cd ../../
yarn develop
```
