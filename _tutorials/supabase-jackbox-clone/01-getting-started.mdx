```bash
# From your `/projects` directory
bun create factbox

# Navigate to the new project directory after set-up
cd factbox
```

```bash{numberLines: 1}
# Start the development server
bun dev
```

Create a `/src/components` folder

Install [shadcn-ui](https://ui.shadcn.com/)

```bash
# Set-up for shadcn-ui component library
bunx shadcn-ui@latest init
```
