# Illustrations

Vector illustrations are used to visually represent concepts, ideas, and actions in a more engaging way than text or
icons alone. They can be used to add visual interest to a page, break up large blocks of text, or provide context for
a concept.

All the illustrations are SVG files that use Tailwind CSS classes to style them. This makes it easy to customize
them to match the color theme of each page.
